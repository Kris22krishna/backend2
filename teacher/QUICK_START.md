# Quick Start Guide - SKILL100.AI Teacher Dashboard

## Get Started in 3 Steps

### Step 1: Install Node.js
Download and install Node.js (version 18 or higher) from [nodejs.org](https://nodejs.org/)

### Step 2: Install Dependencies
Open terminal/command prompt in the project folder and run:
```bash
npm install
```

### Step 3: Run the Application
```bash
npm run dev
```

Open your browser and navigate to: **http://localhost:5173**

## Login to the Application

## Key Features to Explore

### 1. Dashboard
- View overview statistics
- Select different grades (6-10) using tabs
- Use the **Topics dropdown** to filter mathematics performance by specific topics:
  - Fractions, Decimals, Ratios, Percentages
  - Algebra, Geometry, Mensuration
  - Data Handling, Integers

### 2. Assignments Page
- **Click on pie chart segments** to view student lists by completion status
- **Click on bar chart bars** to view students in specific score ranges
- Both interactions open modal dialogs with detailed student information

### 3. Students Page
- Browse comprehensive student performance data
- Filter by grade level
- Download data to Excel

### 4. Attendance Page
- Track daily attendance
- View attendance trends
- Export attendance data

### 5. Settings Page
- Update teacher profile
- Configure notification preferences
- Manage classes
- **Update school details** (new feature!)
